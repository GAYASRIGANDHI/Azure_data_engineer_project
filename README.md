# Azure_data_engineer_project   Banking Domain
End to End Azure data project  using ADF, Databricks ,delta lake, Sql.

This project simulates a banking data pipeline using a transaction dataset.
Data is ingested using Azure Data Factory, processed using Databricks (PySpark),
and stored in ADLS using Lakehouse architecture.

Features:
- Incremental data loading
- Data validation and cleansing
- Fraud detection analysis
- Bronze, Silver, Gold layers
